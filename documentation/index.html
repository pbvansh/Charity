Preparation

First of all, thanks for purchasing this script . I hope you will love the script . Now it’s turn to see Charity live in action.

  • Download script
  • Login to your codecanyon.net/user/meteros account.
  • Go to your downloads page and Download script.
  • After that, script will be downloaded. Now its turn to install it.

Program Installation

You Must have This Programs On your Pc and work with Windows - Linux - OS

First of all, Our Script built with Laravel Framework so you must Have local Server and Composer Program To Use Script.

  • Instal local server or Use your local server Program Some local server you can Use (work with Windows - Linux - OS) xampp wampserver.
  • Instal Composer Program you can Use (work with Windows - Linux - OS) getcomposer.
  • Instal WinRAR or 7zip Program you can Use (work with Windows - Linux - OS) WinRAR 7zip.
  • Now You can Start Installation

Installation

  • Step 1.

    Now extract the zip file which you have downloaded in the previous process, with Your extract file Programs Files. The zip file contains the following files and folders.
  • Use Your Extract file Programs or download Extract file Programs work with Windows - Linux - OS WinRAR 7zip.

  • Step 2.

    Main theme is Charity folder Now Copy Charity folder and Past it in Your Desktop.

  • Step 3.

    Run your local server Program. Then, open your web browser and navigate to Phpmyadmin.
  • Go to Programs in your Pc : Programs - your local server - Open
  • Open your web browser : in new tap go to https://localhost/dashboard (You must Run your local server Program To see this View)
  • In localhost dashboard Click On Phpmyadmin.

  • Step 4.

    Create a new database and name it "Charity", the same that you named it in ".env" file.
  • Click On New (Left on top in Page Phpmyadmin)
  • Now Create database name "Charity" and Click Create.

  • Step 5.

    import database From Files Database. (Charity.sql).
  • Click On your database "Charity".
  • On Top you will see "import" Click On it
  • Now you see File to import Page Click On "Browse" Button and import Our File Database (Charity.sql).
  • Click On Go.

  • Step 6.

    Run Our Project "Charity"
  • Open Charity Folder and Click right on Computer mouse With shift keyboard
  • Click On Your PowerShell
  • In Your PowerShell run command from this directory : php artisan serve
  • Now Charity Project Is Run.

  • Step 7.

  • Open your web browser and navigate to: http://localhost:8000
  • congratulations Your project is View.
  • See This video To learn Installation Script Video Or Follow This Step.

Database

Note : if you want to change database name go to .env file and change name:

APP_DEBUG=true
APP_URL=http://localhost:8000

LOG_CHANNEL=stack

DB_CONNECTION=mysql
DB_HOST=127.0.0.1
DB_PORT=3306
DB_DATABASE=Charity
DB_USERNAME=root
DB_PASSWORD=

BROADCAST_DRIVER=log
CACHE_DRIVER=file
QUEUE_CONNECTION=sync
SESSION_DRIVER=file
SESSION_LIFETIME=120
